What it does

Everything you need to model architecture

Architise covers the full ArchiMate 3 standard across three layers, a dedicated Swimlane mode for process diagrams, and exports compatible with Archi 5.x — all in a single file you can open in any browser.

⬡

Full ArchiMate 3 coverage

Business, Application, and Technology layers — with Active Structure, Behaviour, and Passive Structure element types, and all ArchiMate relationship types.

⟳

Swimlane mode

A dedicated Swimlane editor for process diagrams, completely separate from the ArchiMate workspace. Bridge from Business Layer actors to Swimlane lanes in one click.

⬇

Archi 5.x export

Export your model as a native .archimate file, readable by Archi 5.x and compatible with the Open Group AMEFF format. Also exports to draw.io.

🌐

English & Dutch

Switch between English and Dutch at any time. All tooltips, empty states, onboarding, and the full manual are available in both languages.

📁

Save & load

Save your model as JSON and reload it later. Use Save As to choose filename and folder. Works with the browser's native file picker on modern browsers.

✦

Guided start

An onboarding wizard and ready-made templates help you build your first architecture model in minutes — no prior ArchiMate knowledge required.
